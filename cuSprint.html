CU SPRINT

CU Sprint is a platformer created in the Unity engine for a final project in a software development class at CU Boulder. This class taught me how to better work in a group to create a finished product using Agile.

This project aimed to create a working piece of software that could be deployed to a website that utilized a database to store the completion times of each level in the game using the Agile process. My team had two groups: a web design group, and a game design group. I was the lead of the game design group and the overall project manager for CU Sprint. My tasks as a project manager included time management, weekly check-ins, and group accountability. My tasks in the game design group included level design, game scripting, asset creation, and character animation.
